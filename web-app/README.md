# Web Application - Sentiment Analysis with FHE

Modern Next.js showcase application for the FHE sentiment analysis project.

## Quick Start

1. **Start the Python API server** (in project root):
   ```bash
   python api_server.py
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Features

- Modern dark theme with red/orange accents
- Interactive FHE process explanation (clickable sections)
- Integrated sentiment analysis demo
- Responsive design
- Smooth animations

## Technologies

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS

## Note

Make sure the Python API server (`api_server.py`) is running on port 8000 for the demo to work.
